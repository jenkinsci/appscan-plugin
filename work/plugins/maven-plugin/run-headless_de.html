Falls ein Build keinen Zugriff auf den Desktop benötigt (weil er nur Kommandozeilen-Tools und -Tests verwendet), kann diese Option verwendet werden.

Dies erlaubt Jenkins, den Buildprozess im "headless mode" laufen zu lassen, indem er -Djava.awt.headless=true zu MAVEN_OPTS hinzufügt. Dadurch wird unter OSX vermieden, dass Dock-Icons während des Builds den Fokus erhalten. Diese Option kann in künftigen Versionen auch zur Auswahl geeigneter Slave-Instanzen verwendet werden.
